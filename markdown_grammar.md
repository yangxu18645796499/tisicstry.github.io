# Markdown语法详解

## 标题
使用`#`表示标题，最多支持六级标题。
```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 段落与换行
直接输入文本即为段落。  
**换行可用两个空格加回车。**

## 强调
- **粗体**：`**文本**` 或 `__文本__`
- *斜体*：`*文本*` 或 `_文本_`
- ~~删除线~~：`~~文本~~`

## 列表
- 无序列表：使用`-`、`*`或`+`
```markdown
- 项目1
- 项目2
```
- 有序列表：使用数字加点
```markdown
1. 项目1
2. 项目2
```

## 引用
使用`>`表示引用。
```markdown
> 这是引用内容
```

## 代码
- 行内代码：使用反引号`` `代码` ``
- 代码块：使用三个反引号
```markdown
```python
print("Hello, Markdown!")
```
```

## 链接与图片
- 链接：`[描述](链接)`
- 图片：`![描述](图片链接)`

## 表格
```markdown
| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
```

## 分隔线
使用三个或以上`-`、`*`或`_`
```markdown
---
```

## 任务列表
```markdown
- [ ] 未完成任务
- [x] 已完成任务
```

## 注脚（部分支持）
```markdown
文本[^1]

[^1]: 注脚内容
```

## 内嵌HTML
可直接插入HTML标签。

## 数学公式
Markdown支持LaTeX语法来书写数学公式（部分平台需插件支持）。

- 行内公式：使用`$公式$`
    例如：`$E=mc^2$` 显示为 $E=mc^2$

- 块级公式：使用`$$公式$$`
    例如：
    ```markdown
    $$
    \int_{a}^{b} f(x)\,dx
    $$
    ```
    显示为
    $$
    \int_{a}^{b} f(x)\,dx
    $$
